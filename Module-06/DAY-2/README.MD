## Sourcecode.java:
```
class Vehicle {
    void display() {
        System.out.println("Vehicle Class");
    }
}

class Car extends Vehicle {
    void print() {
        System.out.println("Car Class");
    }
}

class Alto extends Car {
    void show() {
        System.out.println("Alto Class");
    }
}

public class Main {
    public static void main(String[] args) {
        Alto obj = new Alto();
        obj.display();
        obj.print();
        obj.show();
    }
}
```
